# Static Library Project

## Description
This project creates a static library `libmy.a` containing various utility functions. The library is created from a collection of C files, each implementing a specific function.

## Files
- `libmy.a`: The static library containing all compiled object files.
- `main.h`: Header file containing all function prototypes.
- Source files:
  - _putchar.c
  - _islower.c
  - _isalpha.c
  - _abs.c
  - _isupper.c
  - _isdigit.c
  - _strlen.c
  - _puts.c
  - _strcpy.c
  - _atoi.c
  - _strcat.c
  - _strncat.c
  - _strncpy.c
  - _strcmp.c
  - _memset.c
  - _memcpy.c
  - _strchr.c
  - _strspn.c
  - _strpbrk.c
  - _strstr.c
